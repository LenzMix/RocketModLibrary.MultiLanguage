# RocketModLibrary.MultiLanguage
Library for work with SDLangSystem

How to connect Library
===========
> 1. Add constant like 
> ```
>       public const string pluginid = "SDLangSystem";
>       public static SDMultiLangLib.Lib SDLL;
> ```
>
> 2. Add in 'Load' 
> ```
>       SDLL = new SDMultiLangLib.Lib(this);
> ```
